To run the project in Localhost 
Clone the repo
Then inside /gallery-backend 
use-

```bash
npm i
```

```bash
nodemon app.js
```

Then inside /gallery-frontend
use-

```bash
npm i
```
```bash
npm run dev
```

Get your Pexels key and create an .env file in /gallery-backend, use that key key PEXEL_API_KEY='your-api-key'
